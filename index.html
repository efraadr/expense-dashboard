Settings

The Anthropic API blocks browser requests (CORS). You need a tiny Cloudflare Worker proxy.

1. Go to dash.cloudflare.com → Workers → Create
2. Use the cloudflare-worker-proxy.js file provided
3. Paste your worker URL above

Without this, AI features (Digest, Roast, Ask AI) won't work on GitHub Pages. Everything else works fine.
Data Backup
Net worth & budgets are stored in browser localStorage. Export to back up, import to restore on another device.
